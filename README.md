# 🧠 Multi-Model Machine Learning Comparison

![Python](https://img.shields.io/badge/Python-3.x-blue)
![Machine Learning](https://img.shields.io/badge/ML-Scikit--Learn-orange)
![Status](https://img.shields.io/badge/Status-Active-success)

This project implements and compares multiple machine learning classification models to evaluate their performance on a dataset using standard metrics and visualization techniques.

---

## 📖 Overview

The goal of this project is to analyze how different machine learning algorithms perform on the same dataset. It provides a structured pipeline for preprocessing data, training models, and evaluating their performance.

This project is especially useful for:

* Academic lab work
* Understanding ML model comparison
* Practicing real-world ML workflows

---

## 🚀 Features

* 🔹 Train multiple ML models on the same dataset
* 🔹 Compare model performance
* 🔹 Data preprocessing (scaling & splitting)
* 🔹 Evaluation using standard metrics
* 🔹 Modular code structure (easy to extend)

---

## 🧠 Models Used

The following models are implemented:

* Logistic Regression
* Support Vector Machine (SVM)
* Decision Tree Classifier
* Multi-Layer Perceptron (Neural Network)

---

## 📊 Evaluation Metrics

Models are evaluated using:

* Accuracy
* Precision
* Recall
* F1 Score
* Confusion Matrix
* ROC Curve
* AUC Score
